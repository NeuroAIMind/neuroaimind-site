# NeuroAI Mind Lab Website

A simple Next.js website for NeuroAI Mind Lab, ready to deploy on Vercel.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

1. Push this folder to a GitHub repository in your `NeuroAIMind` organization.
2. Go to Vercel and import the repository.
3. After deployment, add your domain:
   - `neuroaimind.org`
   - `www.neuroaimind.org`
4. In Bluehost DNS, point the domain to Vercel.
